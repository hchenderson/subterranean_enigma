# **App Name**: Subterranean Enigma

## Core Features:

- Storyline Selection: Allow users to select and enter one of the three storylines: Archive of Echoes, The Mechanical Well, or The Shrouded Network.
- Interactive Puzzles: Implement interactive React components for each puzzle, handling user input and solution checking.
- Progression System: Track user progress using LocalStorage to remember completed rooms and key fragments.
- AURELIA Narrative Interludes: Trigger AURELIA's narrative responses and hints based on player actions and puzzle completion status.
- Nexus Convergence Unlock: Unlock the final Nexus zone once all three key fragments (ECHOSPIRE, PULSAR-LINEAGE, AURELION-PRIME) are collected.
- AI-Driven Contradiction Detection: Within the Network room's Identity Hash Extraction puzzle, implement AURELIA's occasional lies, requiring the player to use contradiction detection as a tool based on clues from prior puzzles.
- Responsive Design & PWA Features: Ensure the application is fully responsive on mobile devices and implements PWA features (manifest, service worker, add-to-home-screen).

## Style Guidelines:

- Primary color: Muted amber (#D4A373) to evoke a warm, subterranean atmosphere.
- Background color: Dark grayish-brown (#362D26) to create a deep, atmospheric backdrop.
- Accent color: Coral (#FF7F50) used sparingly for important interactions and highlights.
- UI font: 'Inter' sans-serif for UI text, providing a clean and modern feel.
- Puzzle output & AI messages font: 'Source Code Pro' monospace font to enhance the technical aesthetic of puzzle outputs and AURELIA's messages.
- Use minimalistic, custom-designed icons that subtly represent each storyline's theme.
- Implement subtle animations and transitions, such as soft glows in the Archive and glitch effects in the Shrouded Network, to enhance the immersive experience.
